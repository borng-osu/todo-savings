# todo-savings

## Getting Started

Clone this repo.

```bash
git clone git@github.com:team-sharkfin/todo-savings.git
```

Change to this directory.

```bash
cd todo-savings
```

Install dependencies.

```bash
npm ci
```

Rename `.env.example` to `.env`, update the necessary values, then run the `dev` script.

```bash
npm run dev
```
